# Глава 3. Рабочие черновики

Папка содержит рабочие материалы для главы 3 диссертации:
«Проектирование, разработка и экспериментальная оценка прототипа AI-агента».

## Файлы разделов

- `3.1_problem_statement.md` — постановка задачи и описание частично формализуемого бизнес-процесса.
- `3.2_architecture.md` — проектирование архитектуры прототипа.
- `3.3_ml_component.md` — подготовка данных и ML-компонент классификации транзакций.
- `3.4_agent_logic.md` — агентная логика и интеграция генеративных компонентов.
- `3.5_experiment_methodology.md` — методика экспериментальной оценки.
- `3.6_results.md` — результаты эксперимента и анализ эффективности.
- `notes.md` — рабочие вопросы, решения по терминологии и открытые пункты.

## Рабочий процесс

1. Сначала формируется тезисный план подраздела.
2. Затем пишется черновик академического текста.
3. После проверки формулировок текст переносится в финальную версию диссертации.

Экспериментальные показатели будут добавлены после проведения измерений.
